# Beadle

<span class="champIntro">Beadle is unaffiliated, True Neutral and normally benefits from the following stat nodes:</span>

{::nomarkdown}
<div id="champStats">
{:/nomarkdown}
* 13+ Strength
* 13+ Dexterity
* 15+ Dexterity
* 13+ Intelligence

{::nomarkdown}
</div>
{:/nomarkdown}


<span class="champIntro">Beadle is a melee and magic champion.</span>
