# Drizzt

Drizzt benefits from the following stat nodes:

* Dexterity
* Constitution
* Intelligence ![Feat Icon](images\feat.png)
* Wisdom
* Charisma ![Feat Icon](images\feat.png)

Drizzt is a melee champion.
